# @framers/agentos-skills

Curated catalog of **skills** for the AgentOS ecosystem.

Skills are directories containing a `SKILL.md` file with YAML frontmatter (metadata + requirements)
and a markdown body (instructions injected into an agent prompt).

This package is the static catalog (similar to `@framers/agentos-extensions`). For a runtime loader,
use `@framers/agentos` skills utilities and/or `@framers/agentos-skills-registry`.
