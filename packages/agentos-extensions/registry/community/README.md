### How to Contribute

1. Choose a category for your extension
2. Use a template from `/templates`
3. Develop in the appropriate category folder
4. Follow naming convention: `@framers/agentos-{category}-{name}`
5. Submit PR with tests and documentation

### Quality Standards

All community extensions must:
- Have >80% test coverage
- Include comprehensive documentation
- Follow TypeScript best practices
- Pass security scanning
- Be MIT licensed

### Support

Community extensions are supported by their authors and the community.
- Report issues on GitHub
- Contribute fixes via PRs
- Join Discord for help
