### finishReason

Reason the model stopped generating.
- `"stop"`: natural end of response.
- `"length"`: `maxTokens` limit reached.
- `"tool-calls"`: loop exhausted `maxSteps` while still calling tools.
- `"error"`: provider returned an error.
