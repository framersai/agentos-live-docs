Defined in: [packages/agentos/src/api/agent.ts:272](https://github.com/framersai/agentos/blob/369f4181e3a31735ff56401807893a6801760447/src/api/agent.ts#L272)

Sends a user message with a Zod schema enforced server-side via the
provider's native structured-output API. The reply is parsed and
validated; the typed object is returned on `result.object` alongside
the JSON string in `result.text`. Session history is appended just
as in the text-only path, so subsequent `send` calls see the
structured response in their conversation context.

##### Type Parameters

###### S

`S` *extends* [`ZodType`](../@framers/namespaces/z/interfaces/ZodType-1.md)\<`unknown`, `unknown`, [`$ZodTypeInternals`](../@framers/namespaces/z/namespaces/core/interfaces/$ZodTypeInternals-1.md)\<`unknown`, `unknown`\>\>

##### Parameters

###### input

`MessageContent`

User message as text string or MessageContent array.

###### opts

[`SessionSendOptions`](SessionSendOptions.md)\<`S`\> & `object`

Must include `responseSchema`.

##### Returns

`Promise`\<[`SessionSendStructuredResult`](SessionSendStructuredResult.md)\<[`output`](../@framers/namespaces/z/namespaces/core/type-aliases/output.md)\<`S`\>\>\>

Result with a typed `object: z.infer<S>` field.

##### Throws

If the provider returns
        schema-enforced JSON that nonetheless fails Zod validation
        (real provider bug; not retried).
