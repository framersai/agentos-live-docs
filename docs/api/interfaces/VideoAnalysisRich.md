# Interface: VideoAnalysisRich

Defined in: [packages/agentos/src/media/video/types.ts:503](https://github.com/framersai/agentos/blob/7021709ae8e384df5464f1e2ae8b3fca40f72dbb/src/media/video/types.ts#L503)

Rich video analysis result with full scene descriptions, summary,
transcript, and optional RAG chunk references.

This is a richer version of the base [VideoAnalysis](VideoAnalysis.md) type that
includes [SceneDescription](SceneDescription.md) scenes (with cut types, confidence,
key frames), a generated summary, and optional RAG indexing metadata.

## Example

```typescript
const analysis: VideoAnalysisRich = await videoAnalyzer.analyze(request);

console.log(`${analysis.sceneCount} scenes in ${analysis.durationSec}s`);
for (const scene of analysis.scenes) {
  console.log(`[${scene.startSec}s-${scene.endSec}s] ${scene.description}`);
}
```
